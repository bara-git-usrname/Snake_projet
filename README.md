Nom et description du projet:
Jeu Snake avancé (avec niveaux)
Un jeu Snake amélioré avec vitesse croissante, obstacles, bonus, système de score et meilleurs scores locaux. Le projet intègre des animations CSS, du Canvas pour l’affichage du jeu, ainsi qu’une logique avancée avec JavaScript.

Technologies utilisées :
HTML, CSS et JavaScript

Fonctionnalités principales :

-> Création de la structure du jeu et des niveaux en HTML et CSS.
-> Manipulation dynamique des éléments de la page grâce à JavaScript.
-> Implémentation de la logique du jeu : déplacement du serpent, gestion de la vitesse, collisions, obstacles, bonus.
-> Mise en place d’une gameloop fluide pour assurer le fonctionnement continu du jeu.

Lien vers la version en ligne : https://bara-git-usrname.github.io/Mtawa_Bara_snake_avance_projet/

Nouveautés explorées :

-> L’utilisation de l’élément <canvas> pour dessiner les objets du jeu (serpent, pommes, obstacles…).
-> L'utilisation du LocalStorage pour sauvegarder les meilleurs scores des joueurs.
-> La gestion de plusieurs sections dans une même page : le menu principal, les niveaux et le tableau des scores, qui apparaissent ou disparaissent uniquement grâce aux fonctions JavaScript appelées par les boutons.

Difficultés rencontrées :

-> Comprendre toutes les petites étapes nécessaires au fonctionnement complet du jeu.
-> Gérer à la fois la logique interne du jeu et les aspects liés au design et à l'expérience utilisateur.

Solutions apportées :

-> L’apprentissage et l’utilisation du <canvas> combiné à la logique JavaScript ont permis de structurer correctement le jeu.
-> La manipulation du LocalStorage a été un point important pour enrichir l’expérience utilisateur.
-> La recherche, les tests progressifs et la consultation de la documentation m’ont aidé à surmonter les problèmes techniques rencontrés.
